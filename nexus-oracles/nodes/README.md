# Nexus Oracle Node

Allows you to run a Nexus Oracle node to complete jobs requested by the Nexus Oracle network.

## Instructions

### 1. Download

### 2. Install on a Node server

```
npm install
```

### 3. Define `.env` variables

```
cp .env.example .env
```

### 4. Start server

```
npm start
```

## Gas Costs (Transmission)

* Cycle: ≈ 90,000
* Random value: ≈ 60,000
